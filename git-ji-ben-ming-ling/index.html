git基本命令

2018-05-09

新建分支

git branch branch_name

查看本地分支

git branch

查看远程分支

git branch -a

删除本地分支

git branch -d branch_name

如果远程新建了分支本地没有

git checkout --track origin/branch_name

本地创建了分支远程没有

git push --set-upstream origin branch_name

合并分支到master上

git checkout master

git pull origin master

git merge dev

git push
