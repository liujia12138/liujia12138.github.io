vue项目安装sass-loader

2020-05-06 1 min read

安装sass依赖包
npm install sass-loader --save-dev
安装node-sass，因为sass-loader依赖于node-sass
npm install node-sass --save-dev
在build文件夹下的webpack.base.conf.js的rules里面添加配置
{
test: "sass",
loader: ['style', 'css', 'sass']
}
